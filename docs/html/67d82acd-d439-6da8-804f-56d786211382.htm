      ExecutionContext Methods

      Methods

      IsActive Checks if the current execution context is active (i.e., commands should be executed).
      Pop Pops the top execution frame from the stack, returning to the parent context.
      Push Pushes a new execution frame onto the stack with the given condition result.
      Reset Resets the execution context to a fresh state, clearing any remaining frames.
      SwitchToElse Switches the current execution frame to the ELSE branch.
